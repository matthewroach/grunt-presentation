GRUNT

What is GRUNT

  • Grunt is a task-based command line build tool for JavaScript projects
  • Runs on node.js
  • Uses NPM (Node Package Manager) to extended the functionality

Installing GRUNT

  • You use the NPM and install it globally
  • npm install -g grunt-cli

Using GRUNT

  • From the command line : grunt or Windows grunt.cmd

A grunt Project

  • package.json - Used by NPM, store's metadata for projects published as npm modules. This file is used by grunt to list the grunt dependencies needed to run the gruntfile.js on your project.
  • gruntfile.js - Used to configure or define tasks and load Grunt plugins
  • Both these files should live in the root of your project
  • Note: Where these files are located becomes the root of the grunt project, config in the gruntfile.js is relative to where it's located

package.json

  • Can use npm init for automation
  • You could start with a simple file like:
{
  "name": "my-project-name",
  "version": "0.1.0",
}
  • Run npm install first, when dealing with an existing project
  • install gruntplugins using npm install < module > --save-dev - adds the following structure to package.json
  "devDependencies": {
    "grunt": "0.4.0rc7"
  }

gruntfile.js - The 'gruntfile'

  • Generally speaking it's located in the root of your project, along side the package.json file
  • If you try to run grunt and a gruntfile.js file is not found in the current directory, it will keep looking up the folder tree until one is found
  • Contains the following parts
    • The "wrapper" function
    • Project and task configuration
    • Loading grunt plugins and tasks
    • Custom tasks

gruntfile.js - CODE

module.exports = function(grunt) {
	grunt.initConfig({
 
		pkg: grunt.file.readJSON('package.json'),
 
		uglify: {
    	options: {
      	banner: '/*! <%= pkg.name %> <%= grunt.template.today("yyyy-mm-dd") %> */\n'
    	},
			build: {
      	src: 'src/<%= pkg.name %>.js',
      	dest: 'src/<%= pkg.name %>.min.js'
      }
    }

	});
 
	grunt.loadNpmTasks('grunt-contrib-uglify');
 
	grunt.registerTask('default', ['uglify']);
 
});
